隐私政策

《微步》尊重并保护每一位用户的隐私。以下为我们对数据收集与使用的说明：

1. 数据收集

本应用不会主动上传任何用户的内容数据。所有内容（如项目、记录、鼓励语）均存储在您设备本地。

2. 权限说明

本应用可能请求「文件访问」权限，用于数据导出，仅限本地使用。不会请求通讯录、麦克风等与功能无关的权限。

3. 数据安全

数据存储于本地，如您卸载 App，数据将一并清除。请注意定期备份。

4. 第三方服务

为提供更稳定的服务与应用分析，本应用已集成以下第三方服务：

友盟+SDK

  • SDK 名称：友盟应用性能监控SDK
  • 使用目的：提供 APP 应用性能监控与统计分析服务
  • 运营方：友盟同欣（北京）科技有限公司
  • 收集的个人信息类型：设备信息（Android ID / IDFA / OAID / OpenUDID / GUID；可选-IMEI / IMSI / ICCID）、网络信息、位置信息（可选）
  • 隐私权政策链接：https://www.umeng.com/page/policy

友盟+SDK 还可能收集您的设备 Mac 地址、唯一设备识别码（IMEI / Android ID / IDFA / OPENUDID / GUID / IP 地址 / SIM 卡 IMSI 信息），并通过地理位置校准报表数据准确性，提供基础反作弊能力。相关信息仅用于统计分析、性能优化和反作弊校验，不会用于与上述目的无关的用途。

5. 修改与联系

隐私政策可能随版本更新而调整。若有疑问，请通过应用内“联系开发者”功能与我沟通。
